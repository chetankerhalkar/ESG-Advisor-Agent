# Sample ESG Data Files

This directory contains sample CSV and PDF (text format) data for three companies to demonstrate the ESG Intelligence Agent capabilities.

## Companies

### 1. TechCorp Global (Technology Sector)
- **CSV**: `techcorp_emissions_2024.csv` - Quarterly emissions and environmental data
- **PDF**: `techcorp_esg_report_2024.txt` - Comprehensive ESG report with E/S/G performance

### 2. GreenEnergy Solutions (Energy Sector)
- **CSV**: `greenenergy_diversity_2024.csv` - Workforce diversity and safety metrics by department
- **PDF**: `greenenergy_sustainability_2024.txt` - Sustainability report with 100% renewable operations

### 3. Sustainable Manufacturing Inc. (Manufacturing Sector)
- **CSV**: `sustainable_mfg_safety_2024.csv` - Monthly safety incidents and compliance data
- **PDF**: `sustainable_mfg_esg_2024.txt` - ESG performance report with focus on safety and operations

## How to Use

### Via Chat Assistant
1. Open the Dashboard and use the Chat Assistant on the right
2. Say: "Create a new company: TechCorp Global, ticker TECH, sector Technology, US"
3. Upload documents: "Upload the TechCorp emissions CSV and ESG report"
4. Run analysis: "Run ESG analysis for TechCorp"
5. Ask analytics questions: "Show me emissions trend for the last 8 quarters"

### Via UI
1. Navigate to Dashboard
2. Click "Add Company" and fill in company details
3. Go to company detail page
4. Click "Upload Document" and select CSV/PDF files from this directory
5. Click "Run Analysis" to start ESG evaluation
